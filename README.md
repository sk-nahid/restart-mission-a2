**What is JSX, and why is it used?**  
ans: JSX is syntax for write HTML with JavaScript code through React.


**What is the difference between State and Props?**  
ans: State is used to save values inside a component and can change over time. Props are passed from a parent component to a child component.


**What is the useState hook, and how does it work?**  
ans: useState is used to store a value in a React component. It returns the current value and a function that can update that value.


**How can you share state between components in React?**  
ans: State can be shared by passing it from a parent component to child components using props. This is called the lifting state up method.


**How is event handling done in React?**  
ans: Event handling in React is done by calling functions through events like onClick, onChange, and others.
